# Scanner Infrastructure - Quick Start

## 🚀 Запуск за 3 команды

```bash
# 1. Запуск всей системы (включая TP Event Listener)
make up

# 2. Проверка статуса
make status

# 3. Тест TP1 Trailing
make trailing-test
```

**Готово!** Система работает. 🎉
