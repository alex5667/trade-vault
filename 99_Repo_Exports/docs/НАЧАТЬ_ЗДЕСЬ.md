## 📱 Telegram - Что нужно

### Обязательно установить:

```bash
export TELEGRAM_BOT_TOKEN="ваш_токен_от_@BotFather"
export TELEGRAM_CHAT_ID="ваш_chat_id"
```

Или создать `.env` файл:

```bash
cat > .env << EOF
TELEGRAM_BOT_TOKEN=ваш_токен
TELEGRAM_CHAT_ID=ваш_chat_id
EOF
```

### Как получить:

1. **Bot Token**: Напишите @BotFather в Telegram → /newbot
2. **Chat ID**: Напишите боту /start → используйте @userinfobot

## ❓ Частые вопросы

### Как запустить систему?

```bash
make up-bg
```

### Как проверить, что всё работает?

```bash
make check-xauusd-services
```

### Где смотреть логи?

```bash
make tracker-logs
```

### Как протестировать Telegram?

```bash
make test-tracker-telegram
```

### Система не работает, что делать?

```bash
# 1. Проверьте логи
make logs | grep ERROR

# 2. Запустите диагностику
make diagnose

# 3. Смотрите troubleshooting
cat documentation/05_OPERATIONS.md
```

### Нет статистики в Telegram

**Причины**:

1. Не установлены TELEGRAM_BOT_TOKEN или TELEGRAM_CHAT_ID
2. Нет сигналов от сервисов
3. Сервис трекера не запущен

**Решение**:

```bash
# Проверьте Telegram
make check-telegram

# Проверьте трекер
make tracker-status

# Проверьте сигналы
make check-redis-streams
```
