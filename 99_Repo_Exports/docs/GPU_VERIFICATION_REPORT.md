1. **Дождаться активной обработки**:

   - Мониторить во время активной торговли

2. **Проверить во время обработки**:
   ```bash
   # Мониторинг GPU в реальном времени
   watch -n 1 nvidia-smi
   
   # Проверка CPU
   docker stats scanner_infra-multi-symbol-orderflow-1
   ```

3. **Проверить логи во время обработки**:
   ```bash
   docker logs -f scanner_infra-multi-symbol-orderflow-1 | grep -E "(GPU|processing|tick)"
   ```
