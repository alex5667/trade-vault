### Интеграция GPU в код

#### 1. `candle_of_worker.py` (Order Flow обработка)
- ✅ GPU сервис инициализирован
- ❌ **НЕ используется** в текущей реализации
- Причина: обработка свечей по одной через `CandleDeltaDetector.on_closed_candle()`, а не батчами
- GPU сервис доступен, но не вызывается

#### 2. `metrics/features.py` (Метрики)
- ✅ GPU используется для:
  - `zscore()` - если len(series) > 1000
  - `atr_from_bars()` - если len(bars) > 100
  - `cvd_from_delta()` - если len(delta) > 1000
- ⚠️ Условие: данные должны быть достаточно большими

## Вывод

**В данный момент GPU используется частично:**
- ✅ GPU доступен и работает
- ✅ GPU используется в `metrics/features.py` для больших массивов данных
- ❌ GPU **НЕ используется** в `candle_of_worker.py` (обработка свечей по одной)

## Рекомендации

Для полного использования GPU в `candle_of_worker.py` нужно:
1. Реализовать батч-обработку свечей
2. Использовать `gpu_service.process_candles_batch()` вместо обработки по одной
3. Накапливать свечи в батчи и обрабатывать их группами
