## 🔍 Если проблемы

### Ошибка все еще есть?

```bash
# Проверьте, что hotfix применен
grep -A 2 "signal-performance-tracker:" docker-compose.yml | grep "profiles"

# Должно показать:
#   profiles:
#     - default
```

Если не показывает, **обновите docker-compose.yml вручную**:

1. Откройте файл в редакторе
2. Найдите строку `signal-performance-tracker:`
3. Добавьте сразу после нее:
   ```yaml
   profiles:
     - default
   ```
4. Сохраните и запустите: `make down && make up-bg`

### Другие проблемы?

```bash
# Полная диагностика
make diagnose

# Проверка здоровья
make health

# Логи
make logs | grep ERROR
```

## 📱 Telegram (обязательно!)

Перед запуском убедитесь, что установлены credentials:

```bash
# Проверьте
echo $TELEGRAM_BOT_TOKEN
echo $TELEGRAM_CHAT_ID

# Если пусто, установите:
export TELEGRAM_BOT_TOKEN="your_bot_token_here"
export TELEGRAM_CHAT_ID="your_chat_id_here"

# Или создайте .env файл
cat > .env << EOF
TELEGRAM_BOT_TOKEN=your_token
TELEGRAM_CHAT_ID=your_chat_id
EOF
```
